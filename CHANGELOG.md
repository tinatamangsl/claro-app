## [1.0.0] - 2026-03-22

### Added
- 🎧 **Voice Commute Briefing** — Telnyx outbound call + Resemble AI Lucy voice reads top 3 priorities
- 🔄 **ClawdTalk Two-Way Reply** — speak task updates on your commute call
- 🤖 **ClawHouse AI Agent** — persistent Ask Claro floating widget on every page
- 🧠 **Context Modes** — Commute, Deep Focus, Café, Airport
- 🌸 **Cycle Intelligence** — hormonal energy phase alignment for priorities
- ⚡ **Peak Energy Timeline** — colour-coded energy zones with real-time advice
- 🗺️ **Working Memory Breadcrumbs** — visual task history, last 5 completed tasks
- 🌍 **6-Language Voice Support** — English, Polish, Hindi, German, French, Spanish
- 🎯 **SMART/OKR Goals** — goal creation, milestone splitting, and tracking
- ✅ **Eisenhower Task Matrix** — AI-ranked tasks by urgency and importance
- 🔥 **Habit Streaks** — visual progress tracking with celebration animations
- 👥 **Community + Buddy System** — peer accountability hub
- 🔒 **Auth flow** — login page with redirect for unauthenticated users
- 🌱 **Seed/demo data** — app works immediately on first load for demo purposes
- 📱 **Mobile-first responsive design** — glassmorphism with amber gradient system
- ♠ **Accessibility** — neurodiversity-aware prioritisation and scheduling
- 📝 `.env.example` — all 4 sponsor API keys documented
- 📄 `package.json` — project metadata and scripts
- 📁 `backend/` — Express.js + SQLite REST API with Resemble AI voice script generator
- 📁 `assets/` — full frontend source (React + TypeScript + Vite)

### Hackathon
- **🏆 OpenClaw Hackathon 2026 submission** — integrates all 4 sponsors: Telnyx, Resemble AI, ClawdTalk, ClawHouse
- Promo code: `CLAW_HACKATHON_26`
